Class: ManagedServer

ManagedServer

ManagedServer wraps around a Server to add additional features. By wrapping around the server implementation, we gain the ability to shutdown and restart the server when needed. Each time the server is stopped, we destroy all existing socket connections so node does not wait on them to end before calling the close() callback.
